# APARÊNCIA DO ANIMAL (linguagem reforçada 🆕)

| Característica | Detalhe |
|---|---|
| **Disco facial** | Perfeitamente circular, formado por centenas de penas finas organizadas radialmente. Silhueta imediatamente reconhecível como coruja-das-neves. **É a identidade do rosto — prioridade máxima de consistência.** |
| **Olhos** | Amarelo-dourado (#DAA520) ENORMES, frontais. Íris extremamente detalhada, profundidade óptica real, reflexos naturais (catchlight suave). Sensação de inteligência, empatia e escuta — o observador sente que está sendo ouvido. |
| **Plumagem** | Branca predominante (#F5F0F0) com manchas pardas sutis (fêmea). Extremamente volumosa, camadas sobrepostas de penas macias, cada pena individualmente definida, microvariações de branco/creme/cinza-claro. Sensação visual de abrigo — ave adaptada ao Ártico. |
| **Bico** | Pequeno, curvado, preto, quase escondido pelas penas faciais — discreto, não é arma. |
| **Patas** | Emplumadas até os dedos, garras escuras; mãos que seguram documentos com delicadeza. |
| **Postura** | Aberta, ombros relaxados, cabeça levemente inclinada, expressão de escuta ativa. Presença tranquilizadora: não transmite medo nem superioridade — transmite segurança emocional e proteção jurídica. (Leônidas = autoridade; **Sofia = acolhimento**.) |
